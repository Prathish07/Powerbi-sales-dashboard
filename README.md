# Sales Analytics Dashboard using Power BI

## Overview

This project presents an interactive **Sales Analytics Dashboard** developed using **Power BI**. The dashboard provides insights into product sales performance, category distribution, and revenue trends over time. It enables quick analysis of business metrics through clear and interactive visualizations.

The objective of this project is to demonstrate **data visualization, dashboard design, and business intelligence reporting** using Power BI.

---

## Features

* KPI cards showing **Total Sales, Orders, and Customers**
* **Category-wise sales distribution** using pie/donut charts
* **Product-level sales comparison** using bar charts
* **Time-series analysis** of sales by **year and quarter**
* Interactive visualizations for better business insights

---

## Tools and Technologies

* **Power BI**
* Data Visualization
* Dashboard Design
* Business Intelligence Reporting

---

## Dashboard Insights

The dashboard helps in:

* Identifying **top-performing product categories**
* Understanding **sales trends over time**
* Comparing **product performance**
* Monitoring **overall business KPIs**
